Nutrasight

This is by far my most complex project. The site can calculate the nutritional data of a recipe entered by a user and can save it to a database. It uses PHP, JQuery, SQL, JavaScript, along with CSS and HTML. It dynamically looks up a user provided ingredient in a database, returning information from several other tables to allow proper measurements to be calculated.
